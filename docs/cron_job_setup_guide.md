# 定时任务配置指南

## 概述

本文档说明如何在 Coze 平台上配置 AI 新闻推送工作流的定时任务，实现每日早上 9 点自动抓取并推送 AI 资讯。

## 定时任务配置步骤

### 1. 登录 Coze 平台

访问 [Coze 平台](https://www.coze.cn/) 并使用你的账号登录。

### 2. 进入你的项目

1. 进入「工作流」或「Workflow」页面
2. 找到你的「AI新闻推送工作流」项目

### 3. 创建定时任务

#### 3.1 找到定时任务入口

在项目页面中，找到以下任一入口：
- 「定时任务」或「Cron Jobs」
- 「自动化」或「Automation」
- 「调度」或「Scheduler」

#### 3.2 创建新任务

点击「创建定时任务」或「新建任务」按钮。

### 4. 配置定时任务

#### 4.1 基本配置

```
任务名称: AI新闻推送-每日9点
任务描述: 每天早上9点自动抓取最新AI资讯并发布到微信公众号
```

#### 4.2 工作流配置

```
选择工作流: AI新闻推送工作流
```

#### 4.3 定时配置（关键）

**Cron 表达式**: `0 9 * * *`

**Cron 表达式说明**:
```
┌───────────── 分钟 (0 - 59)
│ ┌───────────── 小时 (0 - 23)
│ │ ┌───────────── 日期 (1 - 31)
│ │ │ ┌───────────── 月份 (1 - 12)
│ │ │ │ ┌───────────── 星期 (0 - 6，0和7都代表周日)
│ │ │ │ │
* * * * *
```

**`0 9 * * *` 含义**:
- 分钟：0（整点）
- 小时：9（早上9点）
- 日期：*（每天）
- 月份：*（每月）
- 星期：*（每天）

**效果**: 每天早上 9:00 执行

#### 4.4 输入参数配置

```json
{
  "query": "AI人工智能最新进展",
  "count": 20,
  "news_count": 12
}
```

**参数说明**:
- `query`: 搜索关键词（AI人工智能最新进展）
- `count`: 搜索数量（20条，用于筛选最新内容）
- `news_count`: 最终提取的新闻数量（12条，范围8-15条）

#### 4.5 启用配置

```
启用状态: 开启
```

### 5. 其他常用 Cron 表达式

如果需要调整执行时间，可以参考以下表达式：

| Cron 表达式 | 说明 |
|------------|------|
| `0 9 * * *` | 每天早上9点 |
| `0 9 * * 1-5` | 每周一到周五早上9点 |
| `0 8,12,18 * * *` | 每天8点、12点、18点 |
| `0 */6 * * *` | 每6小时一次 |
| `0 9 * * 0` | 每周日早上9点 |
| `0 9 1 * *` | 每月1号早上9点 |

### 6. 保存并启用

1. 检查所有配置是否正确
2. 点击「保存」或「Save」
3. 点击「启用」或「Enable」
4. 确认任务状态为「已启用」或「Active」

### 7. 验证定时任务

#### 7.1 查看任务状态

在定时任务页面，确认：
- ✅ 任务状态：已启用
- ✅ 下次执行时间：显示正确
- ✅ 上次执行时间：如果有

#### 7.2 查看执行日志

1. 点击任务进入详情页
2. 查看「执行日志」或「Execution Logs」
3. 确认执行情况：
   - ✅ 执行时间
   - ✅ 执行状态（成功/失败）
   - ✅ 执行结果（草稿ID）

#### 7.3 手动测试（可选）

如果不想等到定时执行，可以手动触发测试：
1. 在任务详情页点击「立即执行」或「Run Now」
2. 等待执行完成
3. 查看执行结果

## 新闻时效性说明

### 搜索策略

工作流会自动：
1. 搜索包含"今日"、"最新"关键词的资讯
2. 过滤掉发布时间超过前一日的内容
3. 只保留当日或前一日的最新资讯

### 数量控制

- 搜索数量：20条（从多渠道获取）
- 最终数量：12条（8-15条范围，可配置）
- 确保每条资讯都是最新的

## 监控和维护

### 1. 查看执行日志

定期检查定时任务的执行日志，关注：
- 执行是否成功
- 是否有错误
- 草稿创建是否正常

### 2. 检查微信公众号

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「草稿箱」
3. 查看自动创建的草稿
4. 审核并发布

### 3. 设置告警（可选）

在 Coze 平台配置失败告警：
- 邮件通知
- 短信通知
- Webhook 通知（对接企业微信、钉钉等）

## 常见问题

### Q1: 定时任务没有执行？

**排查步骤**:
1. 检查任务是否已启用
2. 检查 Cron 表达式是否正确
3. 查看执行日志是否有错误
4. 确认工作流配置正确

### Q2: 执行失败怎么办？

**排查步骤**:
1. 查看执行日志和错误信息
2. 检查微信公众号集成是否正常
3. 确认网络连接正常
4. 重新执行手动测试

### Q3: 如何调整执行时间？

**解决方法**:
1. 编辑定时任务
2. 修改 Cron 表达式
3. 保存并重新启用

### Q4: 新闻不够新怎么办？

**解决方法**:
1. 增加 `count` 参数值（搜索更多内容）
2. 调整 `query` 参数（使用更精确的关键词）
3. 检查搜索渠道是否正常

### Q5: 每天只能推送1次吗？

**说明**:
- 订阅号：每天只能群发 1 次
- 服务号：每月可以群发 4 次
- 建议：设置合理的执行频率

## 总结

按照以上步骤配置完成后，你的 AI 新闻推送工作流将每天早上 9 点自动执行：
1. 搜索最新 AI 资讯
2. 筛选当日或前一日的最新内容
3. 生成专业的简报
4. 创建草稿到微信公众号

你只需要登录微信公众号后台审核并发布即可！
